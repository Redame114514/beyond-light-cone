# 光锥之外

一个用于辅助生成 AI 作图 prompt 的静态网页工具。

主题围绕后人类巨构、世界边界、微小人物、场景基底、光线氛围与诗性孤独，通过可编辑表单组合出结构化图像提示词。

在线访问：

https://redame114514.github.io/beyond-light-cone/

## 功能

- 选择画面比例、构图方案、巨构类型、场景基底、人物尺度、光线与情绪。
- 自动生成正向 prompt 和负面 prompt。
- 支持 ChatGPT 图片生成、Midjourney、SDXL/FLUX 等输出预设。
- 支持随机生成和一次生成 10 条变体。
- 所有词库都可以自定义增删改。
- 用户修改会保存在浏览器本地，下次打开仍然保留。
- 支持导出 / 导入配置 JSON，方便分享自己的词库。

## 使用方式

直接打开 `index.html` 即可使用，不需要安装依赖，也不需要后端服务。

如果使用在线版本，直接访问上方链接即可。

## 本地文件

- `index.html`：页面结构。
- `style.css`：界面样式。
- `app.js`：词库、交互逻辑与 prompt 生成逻辑。

## 分享配置

如果你修改了词库，想让别人使用同一套配置：

1. 在网页中点击“导出配置 JSON”。
2. 把导出的 JSON 文件发给对方。
3. 对方打开网页后点击“导入配置 JSON”。

浏览器本地记忆只保存在各自设备上，不会自动同步到其他用户。
