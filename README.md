# 光锥之外

一个用于辅助生成 AI 作图 prompt 的静态网页工具。

主题围绕后人类巨构、世界边界、微小人物、场景基底、光线环境、阴影交互与诗性孤独。v2 版本将流程改为“情绪基底先行”，并加入场景域兼容、天气 / 光源 / 天空 / 阴影硬约束、巨构画面遮挡控制和 SVG 构图预览。

在线访问：

https://redame114514.github.io/beyond-light-cone/

## 功能

- 先选择情绪基底，再由系统推荐画幅、构图、巨构、场景、天气、光源、天空和阴影。
- 拆分天气、光源、天空和阴影，避免太空晴天、月面雨后等物理冲突。
- 使用实际遮挡画面、视觉重量、天空遮挡、阴影覆盖等参数控制巨构存在感。
- 支持低丘、山麓、台地、湿地、盐滩、雪原、月面、近轨、环世界和巨构内部等场景域。
- 提供 SVG 构图预览，显示画幅、三分线、地平线、巨构体块、引导线、人物点位和阴影范围。
- 自动生成正向 prompt 和负面 prompt。
- 支持 ChatGPT 图片生成、Midjourney、SDXL/FLUX 等输出预设。
- 支持随机生成和一次生成 10 条变体。
- 所有词库都可以自定义增删改。
- 用户修改会保存在浏览器本地，下次打开仍然保留。
- 支持导出 / 导入配置 JSON，方便分享自己的词库。

## 使用方式

直接打开 `index.html` 即可使用，不需要安装依赖，也不需要后端服务。

如果使用在线版本，直接访问上方链接即可。

## 本地文件

- `index.html`：页面结构。
- `style.css`：界面样式。
- `app.js`：词库、兼容约束、交互逻辑、构图预览与 prompt 生成逻辑。

## 分享配置

如果你修改了词库，想让别人使用同一套配置：

1. 在网页中点击“导出配置 JSON”。
2. 把导出的 JSON 文件发给对方。
3. 对方打开网页后点击“导入配置 JSON”。

浏览器本地记忆只保存在各自设备上，不会自动同步到其他用户。
